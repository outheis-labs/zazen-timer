# Zazen Timer

A minimal meditation timer for Zazen.  
Ein minimalistischer Timer für Zazen.

**[→ Open App / App öffnen](https://outheis-labs.github.io/zazen-timer/)**

---

## Features

- Configurable Zazen and Kinhin duration
- Multiple rounds with preparation and return times
- Traditional acoustic signals (Han and Gong)
- Custom sound support
- Works offline (PWA)
- Bilingual (DE / EN)

## Funktionen

- Konfigurierbare Zazen- und Kinhin-Dauer
- Mehrere Runden mit Vorbereitungs- und Rückkehrzeit
- Traditionelle akustische Signale (Han und Gong)
- Eigene Sounds verwendbar
- Offline nutzbar (PWA)
- Zweisprachig (DE / EN)

## Install as App

**Important:** Installation is required for reliable timer signals during screen lock. Without installation, the timer may pause when the screen turns off.

### iOS (Safari)
1. Open the app in Safari
2. Tap Share (□↑) → "Add to Home Screen"

### Android (Chrome / Brave)
1. Open the app
2. Tap menu (⋮) → "Add to Home screen" or "Install app"

### Desktop (Chrome / Edge / Brave)
1. Open the app
2. Click the install icon (⊕) in the address bar, or
3. Menu → "Install Zazen Timer"

### Firefox (Desktop)
Firefox does not support PWA installation natively. Use the app in the browser or install the [PWA for Firefox](https://addons.mozilla.org/firefox/addon/pwas-for-firefox/) extension.

## Als App installieren

**Wichtig:** Installation notwendig für zuverlässige Signale bei Bildschirmsperre. Ohne Installation kann der Timer pausieren, wenn der Bildschirm sich ausschaltet.

### iOS (Safari)
1. App in Safari öffnen
2. Teilen (□↑) → "Zum Home-Bildschirm"

### Android (Chrome / Brave)
1. App öffnen
2. Menü (⋮) → "Zum Startbildschirm hinzufügen" oder "App installieren"

### Desktop (Chrome / Edge / Brave)
1. App öffnen
2. Installations-Icon (⊕) in der Adressleiste klicken, oder
3. Menü → "Zazen Timer installieren"

### Firefox (Desktop)
Firefox unterstützt PWA-Installation nicht nativ. Die App kann im Browser genutzt oder mit der [PWA for Firefox](https://addons.mozilla.org/firefox/addon/pwas-for-firefox/) Erweiterung installiert werden.
